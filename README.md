# Running EMQ locally

To install EMQ 2.3.9 image to docker:
```bash
./install.sh
```

To run docker image manually: 

```bash
docker run -tid --name emq -p 1883:1883 -p 8083:8083 -p 8883:8883 -p 8084:8084 -p 18083:18083 emqttd-docker-v2.3.9
```

```bash
docker run -d -p5775:5775/udp -p6831:6831/udp -p6832:6832/udp -p5778:5778 -p16686:16686 -p14268:14268 jaegertracing/all-in-one:latest && open http://localhost:16686
```
